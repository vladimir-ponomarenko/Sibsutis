# Метод простой итерации для решения систем линейных уравнений

Это программа на C++, которая реализует метод простой итерации для решения систем линейных уравнений (СЛУ).

## Описание

Программа позволяет пользователю решить систему линейных уравнений методом простой итерации. Сначала пользователю предлагается ввести количество переменных и количество уравнений в СЛУ. Затем программа запрашивает коэффициенты каждого уравнения. После этого программа решает систему методом простой итерации и выводит решение на экран.

## Как использовать

1. Запустите программу.
2. Введите количество переменных и количество уравнений в системе.
3. Введите коэффициенты каждого уравнения, следуя указаниям программы.
4. Программа выведет решение системы методом простой итерации.

## Примечания

- Программа предполагает, что вводимые данные корректны. В случае некорректных данных программа может работать некорректно или завершиться с ошибкой.
- Для корректной работы программа требует, чтобы количество переменных и количество уравнений были целыми числами.
- Ввод коэффициентов каждого уравнения осуществляется последовательно.
